# 完了メモ

未実装リストにあった戦闘・移動改善は実装済み。

- 勇者入口付近の混雑再現をテスト化し、卵と占有マスで経路探索が詰まらないよう修正した。
- 魔物の通常徘徊はホーム3マス制限を廃止し、接続済み通路の到達可能マスへ進める。
- 攻撃可能な魔物は位置交換しない。非戦闘時の詰まり回避交換は維持する。
- 全勇者出現後に0.5秒の停止時間を入れ、その間は移動・攻撃・追跡を止める。
- 僧侶と聖女を強化し、本人中心の範囲回復と範囲回復演出を追加した。
- 卵は通過可能な床上オブジェクトにし、孵化時は近くの空き通路へずれる。空きがなければ待機する。
- 捕食は捕食側のHPが減っている時だけ発生する。
- コア隣接だけの土はプレイヤー採掘可能扱いにしない。
- 飛び道具被弾後や入口密集時でも、勇者がコア方面へ再探索して進む。
- コア攻撃時のコア点滅、赤い衝撃波、コアHPバー警告を追加した。

検証コマンド:

```bash
npm run assets:check
npm test
npm run build
```
