# アセット購入候補

現行のゲームは48pxセル、俯瞰寄りの地下防衛、魔物・勇者・鉱脈・洞窟タイルが必要。差し替え時は見た目の品質だけでなく、48pxセルへの合わせやすさ、魔物と勇者の種類、商用利用可否、改変可否を優先する。

## 本命: Franuka Complete Fantasy RPG bundle

- URL: https://franuka.itch.io/
- 用途: 全体差し替えの第一候補。洞窟・鉱石・魔物・勇者をまとめて揃える。
- 強み:
  - 16px/32px/48px版を含むパックが多く、現行の48pxセルに合わせやすい。
  - `Mining & Smithing pack` が採掘・鉱脈テーマに合う。
  - `Fantasy RPG Monster pack` は魔物数が多く、敵味方の置き換え元にしやすい。
  - `Fantasy RPG Heroes pack` は勇者側の職業差を出しやすい。
  - `Dungeon asset pack` と `Elemental Dungeons Fire` で地下・炎系の背景や演出を補える。
- 注意:
  - パックごとに収録内容が違うので、購入前に必要なサイズ・方向・アニメーションを確認する。
  - 既存仕様は8方向・6アクションだが、市販素材は4方向や少ないアクションの場合がある。実装時は不足方向をミラー・流用する前提で調整する。
- 採用方針:
  - 最初に購入を検討するならこの bundle。
  - 48px版を優先して使い、足りないアクションは現行の `idle` / `attack` / `cast` / `dig` / `heal` / `eat` に割り当て直す。

## 次点: Dungeons&Pixels - RPG pack 32x32

- URL: https://indie-vova.itch.io/dungeons-and-pixels-starter-pack
- 用途: タイル品質を上げる補強候補。
- 強み:
  - タイルの見た目がまとまっていて、地下ダンジョンの雰囲気を出しやすい。
  - 32pxタイルと32x48キャラクターは、48pxセル内に収めやすい。
- 注意:
  - 魔物・勇者の種類はFranukaほど多くないため、これ単体で全面置換するには不足しやすい。
- 採用方針:
  - Franukaでキャラクターを揃え、タイルの質感だけ追加で強化したい場合に検討する。

## 低コスト候補: Oryx 16-Bit Fantasy

- URL: https://www.oryxdesignlab.com/products/p/16-bit-fantasy-tileset
- 用途: 安価に広い範囲を揃える候補。
- 強み:
  - 24px素材を2倍拡大すると48pxセルに合わせやすい。
  - 英雄、魔物、ダンジョン、地形、エフェクト、UIまで広く入っている。
- 注意:
  - 有名素材なので、見る人にはOryx素材だと分かりやすい。
  - 利用条件としてクレジット表記が必要な場合があるため、購入時のライセンス確認が必須。
- 採用方針:
  - 低コストで一気に統一感を出したい場合の候補。
  - オリジナル感を重視するなら、色替え・加工・他素材との組み合わせを前提にする。

## 補強候補: Admurin Top Down Monsters Mega Pack

- URL: https://admurin.itch.io/mega-pack-top-down-monsters
- 用途: 大型魔物・ボス用の補強候補。
- 強み:
  - 魔物数が多く、アニメーションも豊富。
  - 大型敵の迫力を出しやすい。
- 注意:
  - 128px系の大きい素材なので、現行48pxセルにそのまま入れるには大きい。
  - 通常魔物全体の差し替えより、ボスや大型上位種だけに使う方が扱いやすい。
- 採用方針:
  - Franuka中心で足りない大型魔物だけ追加したい場合に検討する。

## 実装時の確認ポイント

- 購入前に商用利用、改変、ゲーム内配布、クレジット要否を確認する。
- 現行の `assets/pixel/sprites.json` は48px、4フレーム、8方向、6アクション構成。市販素材を使う場合も最終的にはこの形式に変換する。
- 不足方向は左右反転や近い方向の流用で補えるが、勇者の攻撃・採掘・回復は見た目の差が出る素材を優先する。
- 素材差し替え後は `npm run assets:build`、`npm run assets:check`、AGENTS.md記載のテスト一式を実行する。
