# imagegen-v1

このディレクトリは、OpenAIの組み込み `imagegen` で生成した現行ピクセル素材の原画を保持する。

- `actors/`: 5方向 x 7アクションのモンスター・勇者原画。
- `eggs/`: 9種の卵。
- `tiles/`: 地形・鉱脈・土壌の藻7段階・第二進化鉱脈の内側オーラ4フレーム。
- `effects/`: 5種 x 4フレームの通常エフェクトと、橙・青・白のドラゴン炎（開始・中継・終端 x 4フレーム）。
- `items/`: 5種の資源アイコン（砂・水・菌・ミネラル・空気）。
- `debuffs/`: 5種の状態異常。
- `dialogue/`: 2種の会話立ち絵。
- `reference/`: 共通画風の参照画像。
- `manifest.json`: 原画の並び、色違い、生成情報、プロンプト要約。

`npm run assets:build` はこれらのPNGを切り出して48pxアトラスへ合成するだけで、画像を手続き描画しない。

今後の画像追加・更新も `imagegen` を使う。`imagegen` が利用不能、失敗、または上限到達の場合は、JS / Canvas / SVG / 外部素材 / 手描画で代替せず、理由と未完了範囲をユーザーへ報告する。別のCLIやAPIを使う場合は事前承認を得る。
